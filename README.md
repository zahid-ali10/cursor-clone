# Cursor Landing Page Clone

A frontend clone of the Cursor website landing page, built using HTML and CSS.  
This project focuses on layout accuracy, typography, and modern UI styling.

## Live Demo
https://cursor-assignment.netlify.app/

## Tech Stack
- HTML5
- CSS3 (Flexbox, modern CSS features)

## Features
- Clean and responsive layout
- Modern UI inspired by Cursor
- Custom fonts and assets
- Deployed using Netlify
